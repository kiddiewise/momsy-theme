# Momsy Blog Theme

Bu paket WordPress için başlangıç temasıdır.

## İçerik
- mobile-first ana sayfa
- dark / light mod
- özel single post ekranı
- okunma süresi, yorum, görüntülenme altyapısı
- ilgili yazılar ve yazar kutusu

## Kurulum
1. `momsy-theme.zip` dosyasını WordPress > Görünüm > Temalar > Tema Yükle alanından yükleyin.
2. Temayı etkinleştirin.
3. Menüleri atayın.
4. Yazılara öne çıkan görsel ekleyin.
5. Ana sayfa için yazı veya statik ön sayfa ayarını yapın.

## Sonraki geliştirme önerileri
- ajax beğeni sistemi
- gerçek paylaşım ve kaydetme sistemi
- okunma sayısı için bot filtreleme
- uzman etiketi / anne deneyimi etiketi
- performans için local font ve kritik CSS ayrıştırması
